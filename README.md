# mkmaster-journal

本仓库用于归档和发布多个独立的 HTML 页面。所有页面均为静态内容，不依赖后端服务，可直接在浏览器中打开。

## 项目简介

`mkmaster-journal` 是一个以页面归档为主要目的的仓库。它不针对任何特定领域或平台，仅作为存放 HTML 文档的集合。每个页面独立存在，互不干扰，适合用于记录、展示或临时发布内容。

## 目录结构

```
.
├── index.html          # 首页导航（如有）
├── page1.html          # 示例页面
├── page2.html          # 示例页面
└── assets/             # 静态资源文件夹（可选）
    ├── css/
    ├── js/
    └── images/
```

- 根目录下的 `.html` 文件为独立页面。
- `assets/` 目录用于存放页面所需的样式、脚本或图片资源。
- 如有 `index.html`，通常作为页面列表或入口。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可独立运行。
- 页面之间无依赖关系，便于单独查看或引用。
- 归档内容可能随时间更新或调整，不保证长期保留所有历史版本。

## 维护说明

- 本仓库由维护者不定期更新。
- 如需添加或修改页面，可提交 Pull Request 或联系维护者。
- 请勿向本仓库提交与归档目标无关的文件。
